# Real-Time-Holographic-Pose-Tracking-AI

AI-powered real-time holographic pose tracker that creates a neon 3D skeletal clone using OpenCV, MediaPipe, and dynamic motion visualization.

Features

- Real-time human pose tracking
- Neon holographic skeleton effect
- 3D motion visualization
- Dynamic color animations
- Motion trail effects
- AI-based body landmark detection

Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- Matplotlib
